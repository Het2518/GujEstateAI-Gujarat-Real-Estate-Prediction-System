GujEstateAI — Gujarat Real Estate Prediction System
===============================================

This repository contains a scaffold for the GujEstateAI project. Use this structure to develop data pipelines, models, notebooks and a dashboard.

Structure:

- data/ — raw, processed and prediction outputs
- notebooks/ — exploratory and module notebooks
- src/ — reusable modules and training scripts
- models/ — saved models and encoders
- dashboard/ — Streamlit app
- reports/ — plots and SHAP outputs

Place the original ProjectInfo_Gujarat.csv into `data/raw/`.
